# Runtime.Time 属性

命名空间：[Neo.SmartContract.Framework.Services](../index.md)

## 语法

```cs
public static extern uint Time
```

属性值：unix 时间戳，无符号整型。

## 示例

```cs
public static bool Main()
{

        // do something;
    }
}
```
