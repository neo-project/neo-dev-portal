# Neo 介绍


## Neo 是一个由开源社区驱动的区块链平台。

作为功能最全面的区块链开发平台，Neo 开发者可以通过智能合约将资产数字化并进行自动化管理。Neo支持多种编程语言（包括C#、Python、Go、TypeScript和Java），开发者可以轻松地使用他们最熟悉的语言在平台上构建。Neo 还提供了强大的原生基础设施，包括去中心化存储、预言机和去中心化身份系统，为下一代互联网奠定坚实基础。
